# Kii-Cloud-Extension-Server
An extension server for starter pack portal
##部署方式：
 - 安装[Nodejs](https://nodejs.org/en/)环境
 - 安装[mongoDB](https://www.mongodb.org/)数据库
 - 安装sails模块
 
 ```
 sudo npm install -g sails
 ```	
  
 - 安装项目所需模块
 
 ```
 npm install
 ```
 - 启动Mongodb
 
 ```
 mongod 
 ```
 - 启动服务

 ```
 sails lift
 ```
